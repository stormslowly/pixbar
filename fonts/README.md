# Fonts

`JetBrainsMono-Regular.ttf` is sourced from
<https://github.com/JetBrains/JetBrainsMono/releases> (Apache 2.0).
Download it and drop it at this path before running `scripts/build-font.sh`.

`JetBrainsMono-APB.ttf` is the generated patched output — install it
in your OS font manager and `export APB_FONT_PATCHED=1` to unlock the
PatchedSixteenth capability tier.
